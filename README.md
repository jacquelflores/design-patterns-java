## 🚗 Proyecto destacado

Sistema de transporte donde se integran múltiples patrones:

* 🏗️ **Builder** → construcción de vehículos
* 🏭 **Factory** → selección de comportamiento según tipo
* 🧩 **Facade** → simplificación del sistema
* 🔢 **Singleton** → manejo de estado global

💡 Este ejemplo demuestra cómo los patrones pueden trabajar juntos en un escenario más cercano a un sistema real.

---
